Bide Asset Generation Reference
================================

This lists ALL visual assets needed for the game beyond item icons (which are in items.txt).

Style baseline: pixel art RPG, 32x32 or 64x64 icons unless noted, centered on transparent background,
clean outlines, slight shading for depth, no text overlays.
